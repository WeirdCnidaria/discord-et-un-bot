# Et-un bot for Discord

This is a discord bot for simulating card decks and tests for the "Et-un" RPG engine. It adds basic commands for creating decks and doing ability tests. Credit for the engine goes to @kanpka_ on Discord.

## Usage

Make sure you have all the libraries imported in `bot.py` installed. Create a file named `.env` in the project folder, with the Discord app API key. Launch `bot.py` to start the bot.
